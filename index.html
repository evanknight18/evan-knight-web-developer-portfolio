MyPlaylist

MyPlaylist is an interactive web application created using HTML, CSS, and JavaScript coding. It utilizes a musicPlayer sheet to pull song files from your computer, providing a demonstration of the original idea. The application allows you to choose a file, which will autoplay and be automatically added to the playlist. Additionally, the project incorporates the TicketMaster API, enabling users to search for upcoming shows of their favorite artists. Simply type in the names of different artists in the events search bar to explore their scheduled performances.

Professional README Generator

The Professional README Generator is a program built using node.js and the inquirer package. It streamlines the process of creating a high-quality and comprehensive README file for your projects. By simply answering a series of prompts, this program generates a professional README based on your input. It ensures that important sections such as project description, installation instructions, usage guidelines, and contact information are included in the generated README. SVG Image and Logo Generator

The SVG Image and Logo Generator is a project that allows users to generate SVG (Scalable Vector Graphics) files based on their input preferences. By leveraging this tool, users can easily create custom SVG images and logos that can be used in various design projects. The project utilizes Node.js and the Inquirer package to collect user preferences and generate the SVG file accordingly. Whether you need simple shapes or complex designs, this generator provides a flexible solution for creating scalable graphics. To ensure code quality, the project also integrates the Jest package for testing purposes.
